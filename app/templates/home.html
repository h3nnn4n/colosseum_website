Colosseum is a fully automated platform where agents compete agaisnt each other on multiple games. Games run 24/7, non stop. Matches are organized in tournaments, having different formats, and tournaments are grouped in seasons.

Agents can be implemented using whatever technology and programming language that the author wants. It can be a simple script applying an static strategy, it could be using a deep learning model, reinforcement learning, etc. It is even possible to have the logic hosted on a webservice and run it remotly, to get around resource limitations. The only constraints are the rules of the game, which usually have a limit on how fast the agent has to respond.

Sample Agents

We have a collection of sample agents for all the games available on: colosseum_agents. They are MIT licensed. Feel free to modify them and use it as an starting point.
